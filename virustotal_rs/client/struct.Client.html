pub async fn get_popular_threat_categories( &self, ) -> Result<PopularThreatCategoriesResponse>

Get a list of popular threat categories

This returns a list of malware categories commonly used in AV verdicts (e.g., trojan, dropper, ransomware, etc.). The AV verdicts from every file are processed for these categories, normalized (e.g., “ransom” becomes “ransomware”), and set as part of their popular_threat_classification.
